# 学习冲冲冲 - 微信小程序

基于艾宾浩斯遗忘曲线的闪卡学习工具，帮助你高效记忆知识点。

## 功能特性

- ✅ **艾宾浩斯复习算法**：天级别复习间隔（1 天、2 天、4 天、7 天）
- ✅ **数据导入**：支持 CSV/Excel 格式批量导入
- ✅ **标签管理**：按标签分类和筛选卡片
- ✅ **学习统计**：连续学习天数、掌握程度分布
- ✅ **本地存储**：数据存储在本地，保护隐私

## 快速开始

### 1. 打开微信开发者工具

### 2. 导入项目
- 选择目录：`E:\闪卡小程序`
- AppID：选择测试号或填写你的 AppID

### 3. 修改配置（可选）
编辑 `project.config.json`，填入你的 AppID：
```json
{
  "appid": "你的 AppID"
}
```

### 4. 编译运行
点击编译按钮即可预览

## 使用说明

### 导入数据
1. 点击「我的」→「导入数据」
2. 选择 CSV 或 Excel 文件
3. 预览数据后确认导入

### CSV 格式示例

```csv
问题，答案，标签 1，标签 2
"Java 中==和 equals() 的区别？","==比较值或地址；equals() 可重写比较内容","Java 基础","面试高频"
"HashMap 的底层实现？","数组 + 链表 + 红黑树，jdk1.8 后链表长度>8 转红黑树","集合","Java 基础"
```

**注意：** 当问题或答案中包含逗号时，必须用双引号包裹，否则会导致数据解析错误。

### Excel 用户注意

小程序**不支持直接导入 Excel 文件**。请使用以下方法：

1. 在 WPS 或 Excel 中打开文件
2. 点击「文件」→「另存为」
3. 选择格式：**CSV (逗号分隔) (*.csv)**
4. 保存后选择 CSV 文件导入

**标签格式：** 每个标签占一列（从第三列开始），例如：

| 问题 | 答案 | 标签 1 | 标签 2 | 标签 3 |
|------|------|--------|--------|--------|
| 问题内容 | 答案内容 | Java | 集合 | 面试 |

### 手动添加卡片
1. 点击底部「卡片库」
2. 点击右上角「+ 添加」
3. 选择「手动添加」

### 开始学习
1. 首页显示可学习卡片数
2. 点击「开始学习」开始学习
3. 学习后选择掌握程度

## 复习算法

根据掌握程度安排复习间隔：

| 掌握程度 | 间隔天数 |
|---------|---------|
| 生疏    | 1 天    |
| 模糊    | 2 天    |
| 熟悉    | 4 天    |
| 熟练    | 7 天    |

## 数据导出

点击「我的」→「导出数据」，可将所有卡片和学习记录导出为 CSV 文件。

## 注意事项

1. CSV 格式中，含逗号的内容必须用双引号包裹
2. Excel 用户请在 WPS/Excel 中另存为 CSV 格式后导入
3. 数据存储在本地，更换设备需要重新导入
4. 定期使用导出功能备份数据

## 技术栈

- 微信小程序原生开发
- 本地 Storage 存储
- 艾宾浩斯遗忘曲线算法

## 开发说明

### 添加新页面
1. 在 `pages/` 目录下创建页面文件夹
2. 在 `app.json` 的 `pages` 数组中添加路径

### 修改复习间隔
编辑 `utils/review.js` 中的 `REVIEW_INTERVALS` 数组：
```javascript
const REVIEW_INTERVALS = [1, 2, 4, 7] // 可自定义天数
```

### 修改掌握程度选项
编辑 `app.js` 中的 `masteryLevels` 配置。
